C-Programming

C programming is a high-level and general-purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed by Dennis Ritchie at Bell Telephone Laboratories to Develop the Unix Operating System in the early 1970s. Ranked among the most widely used languages, C has a compiler for most computer systems and has influenced many popular languages, Java programming language is also equally popular and most widel y used among modern software programmers.

  1. C Data Type :

    2. C language has some predefined set of data types to handle various kinds of data that we can use in our program.
    Data type of variable gives two kind of information of variable data.
    What is the size of data?
    What is the type of data?

    It helps compiler to reserve memory for variable and how to read and write them.

    C language supports 2 different type of data types:

    Primary data types:
    These are fundamental data types in C namely integer(int), float(float), character(char) and void.

    Derived data types:
    Derived data types are nothing but primary datatypes but a little twisted or grouped together like array, stucture, union and pointer. These are discussed in detai ls later.

  2. Variable & Constant :

    3. Variable : A variable is a named memory location which temporarily stores data that can change while the program is running.

    Constant : Constants refer to fixed values that the program may not alter during its execution. These fixed values are also called literals. Constants can be of any of the basic data types like an in turn, they are categorized into two major categories that are primary constants and secondary constants. In primary constants, we have integer constants, real constants, and character constants, etc. In secondary constants, we have Array, structure, union, pointer, etc.

    register typedef return union
    switch break case for
    if else do while
    default goto sizeof volatile
    continue enum static extern

    example :

    int age ;

    Here, int is keywords.

    Identifier : The name of a variable is known as its identifier.

    int age ;
float calculation;

    Here, age and calculation are identifiers.
    keep remember, identifier names must be different from keywords. You cannot use float as an identifier because float is a keyword.

  5. C Operators :

    6. An operator is a symbol that operates on a value or a variable.
    Operators can be defined as basic symbols that help us work on logical and mathematical operations.

    List of C Operator

    1. Arithmetic Operators :-

    In arithmetic operator is addition (+), subtraction(-), multiplication(*) and division(/), and they are performs mathematical operations on numerical value(int , float , double )

    // Working of arithmetic operators
#include < stdio.h >
int main()
{
	int a,b, result;
    
        result = a+b;   // addition operation
        result = a-b;  // subtraction operation
	result = a*b;  // multiplication operation
	result = a/b;  // division operation
	return 0;
}

    2. Relational Operators :-

    Relational operators find the relation between two operands. i.e. to compare the values of two variables.

    Operator Example
    < a < b (a is less than b)
    > a > b (a is greater than b)
    >= a >= b (a is greater than or equal to b)
    <= a >= b (a is less than or equal to b)
    == a == b (a is equal to b)
    != (a is not equal to b)

    3. Logical Operators

    Logical operators are used to perform logical operations.
    Following Three logical operator,They are, logical AND (&&), logical OR (||) and logical NOT (!).

    Operator Example
    &&(AND) (x>5)&&(y <5) It returns true when both conditions are true
    || (OR) (x>=10)||(y>=10) It returns true when at-least one of the condition is true
    ! (NOT) The NOT operator is a Boolean operator that returns TRUE or 1 when the operand is FALSE or 0, and returns FALSE or 0 when the operand is TRUE or 1.
